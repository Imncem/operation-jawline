# Operation Jawline

Operation Jawline is a tactical discipline and training companion built with Flutter.
The app combines daily check-ins, intelligent workout recommendations, and a gamified progression system to help users maintain consistent physical discipline.

The experience is designed like a training mission system, where daily actions contribute to rank progression and long-term training records.

**Core Concept**

Operation Jawline transforms daily discipline into a structured mission loop:

Check-In → AI Recommendation → Workout Protocol → Debrief → XP Progression → Repeat

The system rewards consistency through a discipline chain, experience points, and rank promotions.

**Main Features**

`Mission Control Dashboard`

-Agent status overview

-Body mass, hydration, energy levels

-Discipline chain (streak tracking)

-Rank and XP progression

-Daily mission progress

`Daily Check-In`

Users submit daily status including:

-Energy level

-Sleep hours

-Water intake

-Mood

-Optional weight

-Face puffiness rating

The check-in contributes to daily mission completion and XP progression.

**ZENITH AI Coach**

A rule-based workout planning system that generates daily training protocols based on user readiness.

Features include:

-Readiness scoring

-Training lane selection (Recovery / Light / Moderate / Hard)

-Workout focus selection (Strength / Mobility / Mixed / Conditioning)

-Safety guardrails

-Explainable recommendations

**Protocol Active (Workout Runner)**

Interactive workout execution with:

-Step-by-step guidance

-Timed and repetition-based exercises

-Rest timers

-Pause, skip, and abort controls

-Progress tracking

-Sound and haptic feedback

**Protocol Debrief**

After completing a session, the system displays:

-Completion percentage

-Time on target

-Protocol effort score

-XP earned

-Promotion triggers if applicable

**Rank & Progression System**

Users progress through a rank ladder:

Recruit → Private → Lance Corporal → Corporal → Sergeant → Staff Sergeant → Warrant Officer → Elite Operator

XP is earned through daily mission completion and workout effort.
